Notes on building a website

Prompt:

  • Build a portfolio website that can showcase HTML CSS and Javascript knowledge
  • Do not use a popular framework like bootstrapper
  • It must be interactive, load quickly and look modern (make use of parallax, multy-column layout, responsive)
  • Must work on mobile
  • If something seems tough, do it

While I worked with html in some form or another for years most of the time when building a quick demo I'd resort to using a framework like Bootstrapper or modify a web template to suit my needs. In order to learn more about HTML and CSS I decided to write my own portfolio website from scratch.

I started by looking at a few different ways to achieve a responsive easy to modify website, the CSS makes use of two different approaches to styling a multi-column layout 1. the projects part makes use of the columns css property 2. the footer makes use of a grid view
